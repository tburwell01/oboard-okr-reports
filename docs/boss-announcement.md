# Corporate OKR Board generator — what to expect

*A short, non-technical message you can forward to your boss / teammates.*

---

**Subject: Quick setup — Corporate OKR Board generator**

Hi [Name],

We've built a small tool that pulls our live FY27 OKR data from Oboard and generates the Corporate
OKR Board one-pager, saving it straight to our SharePoint OKR folder. You can run it yourself in about
a minute once it's set up.

**You won't need to do anything technical** — Claude will set it up on your laptop for you.

**What to do:**
1. **Create a free GitHub account** (if you don't already have one): go to
   **https://github.com/signup**, sign up with your email, and pick a username.
   **Then send me your GitHub username** so I can give you access to the project.
2. Accept the GitHub invite I'll send once I have your username (just click **Accept invitation**).
3. Open **Claude** on your work laptop (the one with file/terminal access).
4. Go to this page, copy everything on it, paste it into Claude, and say
   *"Please follow these instructions to set up the OKR web app on my machine"*:
   **https://github.com/tburwell01/oboard-okr-reports/blob/main/SETUP-WITH-CLAUDE.md**
5. Claude will walk you through the few clicks it needs from you (signing into GitHub, clicking
   **Sync** on our SharePoint OKR folder, and pasting the Oboard token I'll send you separately).

**Once it's running:** a small page opens in your browser with three buttons —
- **Test Run** = preview the latest numbers (safe, run anytime)
- **Next Run** = the official monthly update
- **First Run** = only the very first time

Click one, wait ~20 seconds, and the dated deck appears in our SharePoint OKR folder.

Two requirements: your laptop needs **desktop PowerPoint** installed, and you need **Claude with file
access**. If our security software blocks it, Claude will tell you exactly what to forward to IT.

Thanks!
